A simple and fun web-based game where the player tries to guess a random number between 1 and 10.  
Built entirely with **HTML**, **CSS**, and **Vanilla JavaScript**.

---

##  How to Play

1. Enter a number between **1 and 10**  
2. The game will tell you if your guess is **too high** or **too low**  
3. Keep guessing until you find the **correct number**  
4. Click **"Play Again"** to restart the game and generate a new number  

---

##  Features

-  Custom alert modals (no default `alert()` dialogs)
-  Responsive layout (works on mobile and desktop)
-  Helpful hints for high or low guesses
-  **Play Again** button to reset the game
-  Attempt counter to track how many guesses you made

---

##  How to Run

No build tools, no dependencies.

1. Clone or download this repository  
2. Open `index.html` in your browser  
3. Start guessing!  

You can also host it on any static server (GitHub Pages, Netlify, Vercel, etc).
